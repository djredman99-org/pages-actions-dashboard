# Multiple Dashboards - Quick Start Guide

## What's New?

Your GitHub Actions Dashboard now supports multiple dashboards! Organize workflows by environment, team, or any way that makes sense for your organization.

## Quick Guide

### üîÑ Switch Between Dashboards

1. Look at the top of the page (below the title)
2. Click the dropdown menu
3. Select the dashboard you want to view
4. The page will automatically refresh with the new dashboard's workflows

### ‚ûï Create a New Dashboard

1. Click the **"Manage"** button next to the dashboard selector
2. Enter a name in the "Create New Dashboard" field
3. Click **"Create"**
4. Use the dropdown to switch to your new dashboard
5. Add workflows using the **"Add Workflow"** button

## Data Migration

Don't worry! If you already have workflows configured:
- Your existing configuration will automatically migrate
- All your workflows will be preserved
- They'll be in a dashboard called "Main Dashboard"
- You can rename it or create additional dashboards
