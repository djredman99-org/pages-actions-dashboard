Dashboard Features

This dashboard provides real-time monitoring of GitHub Actions workflow statuses across multiple repositories with enterprise-grade security.

Key Features:

  • Azure Function backend for secure API calls
  • GitHub App authentication with credentials stored in Azure Key Vault
  • Workflow configurations managed in Azure Storage
  • Manual refresh for instant updates
  • Auto-refresh every 5 minutes
  • Support for public, private, and internal repositories

Getting Started

To set up your own instance of this dashboard, follow the comprehensive setup guide in our repository.

Visit the README for complete setup and configuration instructions.

Technology Stack

Built with modern web technologies and Azure cloud services:

  • Frontend: Pure HTML, CSS, and JavaScript
  • Backend: Azure Functions (Node.js)
  • Security: Azure Key Vault, GitHub App Authentication
  • Storage: Azure Storage for configuration management
  • Hosting: GitHub Pages
